## System Overview

**Tagline:** Modular · Goal-driven · Class-compatible

**Core philosophy:** Pick one goal block. Commit 6–12 weeks. Use classes as fixed anchors and self-program the gaps. Don't repeat the same block for more than 10–12 weeks — the variation itself is part of the stimulus.

### How to Use This System

| Parameter | Guideline |
|---|---|
| Endpoint | Race, sim, checkpoint — any works |
| Block length | 6–12 weeks. Don't repeat same block 12+ weeks |
| Gym classes | Fixed anchors — build around them |
| Self-programmed | Long run + interval run + squat day |

---

## Non-Negotiables — Every Week

**Baseline:** 2 runs + 1 squat + 1 metcon. Maintainable across every block without accumulating fatigue. Everything else is optional volume added when recovery is good.

| Session | Description |
|---|---|
| Long Z2 Run | 45–80 min easy, fully conversational |
| Quality Run | Interval or tempo (or pace-targeted HYROX class) |
| Leg Strength | Squat program (RSP or velocity-based) |
| Hard Conditioning | HYROX class or CrossFit class |

## Block Structure

### Block A — Olympic Lifting Focus
**Duration:** 6–10 weeks  
**Goal:** Snatch / C&J PR, front squat strength  
**Emphasis:** Technical skill + explosive power. Lower metcon stress to keep CNS fresh for lifting.

**Weekly structure:**
- Oly sessions: 2–3×/week via class or Weightlifting.ai block
- Squat day: 1× self-programmed, front squat priority — unless already covered inside Weightlifting.ai
- Long run: 1× Z2, 45–60 min (non-negotiable — protect the base)
- Quality run: Reduce intensity by 10–15 s/km OR swap for easy 30 min jog
- Metcon: Optional — short, upper-body bias, low leg stress

**Pull back on:** Long compromised runs, heavy carries, HYROX-specific stations. These fry the legs you need fresh for lifting. Keep conditioning sessions short and aerobic.

**Squat program options:**
- RSP 2×/week — on back or front squat. Don't run RSP 3×/week alongside full Oly — too much CNS.
- Velocity-based — better if Oly class volume is already high; lower residual fatigue.
- Weightlifting.ai — strong option for 6-12 week Oly-focused blocks. Let it be the main lifting anchor; do not layer extra squat volume on top if the program already includes squats and pulls.

**Block goals:** Snatch PR · C&J PR · Master catch position · Front squat 1RM up

---

### Block B — HYROX Prep
**Duration:** 8–12 weeks  
**Goal:** Race-ready, pace discipline  
**Emphasis:** Running economy + power endurance + station-specific fitness. Race or sim as endpoint.

**Weekly structure:**
- Runs: Runna 2×/week plan, full progression (interval + long run)
- HYROX class: Primary metcon anchor — technique → pace discipline → race sim
- Squat day: 1× self-programmed, HYROX accessories (lunges, carries, wall balls)
- Extra Z2: Optional easy bike/row on recovery days

**Key stimulus — Compromised runs every 2–3 weeks:** Either your HYROX class or interval session becomes a circuit of run + stations (e.g. 3×500 m run + 20 wall balls + 40 m lunge). This is the core Block B adaptation.

**HYROX class by phase:**
- Early weeks (Technique): Show up and move well. Runs should feel easy (6:00–6:30/km). "Can I do 8 rounds and survive?"
- Mid weeks (Pace Awareness): Push runs toward target HYROX pace. Stations should hurt, but runs should feel "controlled fast." "Can I hold this pace when legs are cooked?"
- Late weeks (Race Sim): Target runs at exact race pace. If you deviate ±10 s/km, note it as data for race strategy. "This run IS the race."

**Squat program:**
- Velocity-based: Best fit — submaximal load, fully compatible with full Runna + HYROX classes
- RSP: Too recovery-intensive to pair with full running + class volume

**Block goals:** Sub-X HYROX time · Hold pace every split · No station stops · Sled push unbroken

---

### Block C — General Strength + Fitness
**Duration:** 6–10 weeks  
**Goal:** Squat PR, aerobic maintenance  
**Emphasis:** No race, no Oly peak. Strength gains + aerobic maintenance. Great as a reset or dedicated strength cycle between blocks.

**Weekly structure:**
- Strength: 1–2× heavy compound day (back squat, deadlift, press)
- Skill work: 10–15 min fresh at the start: pull-ups, push-ups, bar muscle-up drills, handstands, toes-to-bar
- Classes: CrossFit or functional bodybuilding as conditioning
- Runs: Runna 2×/week, or just long run + easy jog if reducing load
- Optional: Extra class or Z2 when recovery is good

**Squat program options:**
- RSP 2×/week: Best if explicit goal is a squat PR. Reduce Runna intervals to easy jogs during the 6-week RSP cycle.
- RSP 3×/week: Most aggressive option. Keep running to minimum — long run + easy Z2 only.
- Velocity-based: Better if keeping running and conditioning quality high. Run as single squat day alongside full Runna + classes.

**Block variants:**
- Strength Peak: RSP-led block. Minimal skill work. Goal = squat PR.
- Balanced Fitness: Velocity squats + full Runna + classes. General all-rounder.
- Skills + Strength: Skill-first on strength day, then squat, then accessories. Great for pull-ups, push-ups, BMU, HSPU progress.

**Block goals:** +5–10 kg squat 1RM · Maintain running base · Structural balance · Gymnastics skill progress

---

### Block D — Deload / Reset
**Duration:** 2–4 weeks  
**When:** Post-race or post-hard-block  
**Emphasis:** Volume -30–50%. Keep intensity moderate. 2–4 weeks max.

**Weekly structure:**
- Long run: Shorter — 40–50 min easy, conversational
- Quality run: Skip or swap for easy 30 min jog
- Squat: Light session only — RPE ≤6, technique focus
- Class: 1× any class, fun, no ego
- Other: Extra Z2, mobility, walks — whatever feels good

**Goal:** Stay active, recover, and decide what the next goal block is. Don't extend beyond 4 weeks — detraining begins. Two red weeks in a row = you need a deload.

## Example Block Chain

```
💪 Block C — Strength Base      6 weeks · RSP 2×/week · squat PR goal
🔄 Block D — Deload             2 weeks · recover + reassess
🏁 Block B — HYROX Prep        8 weeks · velocity squats + full Runna
🏆 Race or Sim                  Test your fitness
🔄 Block D — Reset              2 weeks
🏋️ Block A — Oly Focus          8 weeks · RSP 2×/week on front squat
```

No race booked? Cycle C → A → C with a HYROX sim every 4–6 months.

## Squat Programs

### Russian Squat Program (RSP)
**Type:** High frequency peaking program — designed to drive a 1RM PR  
**Frequency:** 2× or 3×/week  
**Duration:** 6 weeks → deload  
**Stress:** High  
**Lift:** Back or front squat

**Block fit:**
- Best for Block C (squat PR cycle) or Block A (front squat, 2×/week only)
- 3×/week: Reduce running to minimum — long run + easy Z2 only. No hard intervals.
- 2×/week: More manageable. Compatible with Oly classes + light running. OK in Block A.
- Avoid pairing with full Runna plan + HYROX classes simultaneously

### Velocity-Based Program
**Protocol:** 5×5 @ 55–60% (fast) · 1×1 @ 80% · weekly RM ladder testing  
**Frequency:** 1×/week  
**Duration:** 6–8 weeks  
**Stress:** Low–Moderate  
**Focus:** Speed + power output

**Block fit:**
- Best for Block B (HYROX) or Block C (maintenance while running)
- Fully compatible with Runna 2×/week + HYROX classes
- Why it works: Speed focus trains power output on fatigued legs — exactly what HYROX demands
- RM testing: Weekly ladder (10RM → 8RM → ... → 1RM) gives clear progress signal

### Pairing Matrix

| Program | Block B | Block A | Block C |
|---|---|---|---|
| RSP 3×/week | ❌ | ❌ | ✅ |
| RSP 2×/week | ⚠️ | ✅ | ✅ |
| Velocity | ✅ | ✅ | ✅ |

## Running

### Runna Plan
Runna's HYROX 2×/week plan (1 interval/tempo + 1 long run) maps directly onto the two run non-negotiables. Sustainable across all blocks with per-block adjustments.

| Block | Adjustment |
|---|---|
| Block B | Run as written — full progression |
| Block C | Run as written; dial back interval pace if squat loading peaks |
| Block A | Keep long run; reduce interval intensity 10–15 s/km or swap for easy 30 min jog |
| Block D | Long run only (40–50 min easy); skip interval session |

### Maintenance Base (from non-negotiables alone)
| Session | Distance |
|---|---|
| Long run | 8–12 km |
| Interval session | 5–7 km incl. warm-up/cool-down |
| HYROX class | ~3–5 km |
| **Weekly total** | **~16–24 km/week** |

When HYROX classes drop out (Oly block), swap class running for a short 20–30 min easy jog to keep legs ticking.

### Does HYROX Class Running Count?
| Effort level | What it counts as |
|---|---|
| Deliberately pacing hard | Quality run — can sub your interval session that week |
| Controlled / moderate | Aerobic volume only — contributes to weekly km |
| Just surviving | Z2 / bonus volume only |

Only replace your dedicated interval session with class running if you're intentionally targeting pace. Otherwise keep both.

### Interval Run Progression (Block B)

**Weeks 1–4 · Base**
- Wk 1: 6×400 m @ ~5:15–5:20/km · 90–120 s jog recovery
- Wk 2: 8×400 m same pace · 90 s jog
- Wk 3: 5×600 m @ ~5:10–5:15/km · 2 min jog
- Wk 4: 4×800 m @ ~5:10/km · 2–3 min jog

**Weeks 5–8 · Build**
- Alternate: 6×600 m @ ~5:00–5:05/km · 2 min jog
- Alternate: 5×800 m @ ~5:00/km · 2–3 min jog
- Every 3rd week: Compromised: 3–4 rounds of 500 m run + 20 wall balls or 40 m lunges · walk 2–3 min
- Target pace: ~10–15 s/km faster than your intended HYROX race pace

**Weeks 9–11 · Peak**
- Option A: 4–5×800 m @ ~4:55–5:00/km · 2–3 min jog
- Option B: 3–4×1 km @ target race pace · 3 min jog
- Intent: Lock in race pace. Validate: "Can I hold this after sleds and carries?"

**Week 12 · Taper**
- 2–3×400–600 m @ race effort · long easy warm-up and cool-down

### Long Run Progression (Block B)
| Phase | Protocol |
|---|---|
| Weeks 1–4 | 45–50 min easy, +5–10 min/week to 60–70 min |
| Weeks 5–8 | 60–75 min. Option: last 10–15 min at steady pace (race pace +20–30 s/km) |
| Weeks 9–11 | Two weeks at 70–80 min peak; one week shorter at ~50–60 min |
| Week 12 | 35–45 min easy only |

## Weekly Check-In

### Non-Negotiables Checklist
- [ ] Long Z2 run completed (45–80 min easy, conversational)
- [ ] Quality run completed (interval, tempo, or pace-targeted HYROX class)
- [ ] Squat / leg strength session (per current squat program)
- [ ] Hard conditioning session (HYROX class, CrossFit, or metcon)

### Load Check
- [ ] Fewer than 3 "I was smashed" days (if 3+, trim one session next week or swap for Z2)
- [ ] Quality runs hit their target stimulus (right pace, right effort, right structure)
- [ ] No two consecutive hard sessions (hard day → easy/rest day between)

### Week Colour
- **Green week:** Everything feels controlled at RPE ≤6. Add reps, shave 5–10 s/km off pace, or add 5–10 kg to squat. Proceed with confidence.
- **Yellow week:** One session hit RPE 8–9, others were fine. Hold volume and pace. Focus on cleaner execution, not more load. Happens when life stress, sleep, or travel add up.
- **Red week:** Accumulated fatigue: heavy legs, sluggish runs, poor sleep recovery. Cut volume 30–50%. Keep everything at RPE ≤7. One hard session max. Two red weeks in a row? Reassess the plan.

## Program Design Rationale

**Minimum viable dose:** 2 runs + 1 squat + 1 metcon gives enough exposure to maintain running, strength, and conditioning without turning every week into a recovery problem.

**Modular blocks:** Rotating emphasis blocks lets you progress one quality without peaking everything at once. Don't run the same block for more than 10–12 weeks — the variation itself is part of the stimulus.

**Classes as anchors:** Classes are fixed stressors. Self-programming fills gaps, not competes. This prevents you from doubling up hard sessions on adjacent days.

**Quality filter for HYROX class running:** Class running counts toward volume, but only replaces an interval run when you intentionally target pace on every split.

**Velocity squats in HYROX blocks:** Submaximal load (55–60% 1RM) with fast bar speed preserves neuromuscular power with less residual fatigue than heavy slow reps — meaning you can run Runna's full plan without compromise.

**Skill before strength:** Bodyweight and gymnastics skill work belongs early in the session while CNS is fresh. After heavy squats or a long run, movement quality degrades and so does skill acquisition.

**Oly block caution:** Technical lifting has high neural demand. Running intensity and metcon volume need to drop so CNS resources go toward skill and power expression in the lifts.

## Training Tips

### Recovery Separation
- 24+ hours between hard endurance and heavy strength yields better strength adaptations. If you must stack on the same day, do strength before endurance and fuel aggressively in between.
- Run club → next-day squats: Fine if squats stay lighter (RPE ≤7). Heavy build days are better 48 hrs after a hard interval session.
- HYROX metcon → next-day squats: Light squats (3×3 RPE 7) are manageable. Avoid heavy builds — wait 48 hrs.
- Limit to 2–3 genuinely hard sessions per week (e.g. interval run, HYROX class, and one heavy strength day). Everything else should be easy.

### Fueling Around Concurrent Sessions
- Stacking endurance + strength same day: Fuel hard between sessions. Residual glycogen depletion from a run will tank your strength. Get carbs + protein in the gap.
- Example: Interval run (6×600 m) → 200 kcal carb snack + electrolytes + 30 min rest → Thursday light squats is OK. Thursday heavy squats is risky without adequate refuelling.
- Fatigue-first squat days (Z2 jog then squats): Eat light before the jog (e.g. banana), then fuel properly for the main squat lift. You want pre-fatigued legs, not a glycogen-depleted engine.

### Race Pace Anchor
- Estimate your HYROX pace by Week 5–6. Run 3×1 km at a hard-but-controlled effort. Average the splits. Add 15–25 s/km to get your realistic HYROX race pace target.
- Example: You average 4:58/km unsupported → target HYROX race pace ≈ 5:13–5:23/km → use 5:15/km as your interval anchor in Block B.
- Your interval sessions should be ~10–15 s/km faster than race pace to build a speed reserve. In peak weeks, run repeats at or near race pace to lock it in.
- Revisit every 3–4 weeks. If unsupported pace drops, your race pace anchor drops too.

### Form Gates — When to Stop a Set
- Squat: Stop if knees cave inward, butt tucks under, or bar drifts forward. Rest 2–3 min. Don't grind through degraded reps.
- Lunge / Step-up: Stop if front knee drifts well past toes, trailing knee bounces off the floor, or torso collapses forward.
- Interval run: If stride shortens and cadence drops late in a rep, dial back pace by 10–15 s/km. Finish the session strong.
- HYROX metcon: If wall balls lose target height, or farmer carry lists severely to one side, finish the round then reassess pacing.
- For fatigue-resistance setups (Z2 → squats, HYROX → runs): keep load at 60–75% 1RM. Use volume, not weight, to create the stimulus.

### Taper Week (Race Week)
- Philosophy: Maintain quality of movement and pace. Slash volume by 40–50%. You should feel sharp, not stale.
- Mon: 2–3×400 m at race pace. Long easy warm-up + cool-down. No metcon.
- Tue: 3×3 squat at RPE 6 (light, move fast). Light accessories only — no heavy carries or step-ups. Optional 15 min easy Z2 bike if legs feel flat.
- Wed: Easy jog 20–30 min. Station drills (SkiErg pacing, wall ball rhythm) at 60% effort.
- Thu: Rest or 15 min easy jog + stretching.
- Fri: 1–2×800 m at race pace (controlled, not all-out). Optional: 3 rounds of 500 m run at race pace + 1 station at 70% effort + easy transition walk.
- Race day: Warm-up jog, station drills, go.

### Multi-Race Strategy
- A-Race + B-Race: Use the B-Race as a fitness checkpoint, not all-out effort. Save peak pacing strategy for the A-Race.
- Suggested structure: Weeks 1–6 Build → Week 7 Light taper for B-Race → Week 8 Post-B-Race deload (50% volume) → Weeks 9–11 Peak for A-Race → Week 12 Taper.
- No race booked? A HYROX sim every 4–6 months in Block C or B is enough to test fitness and maintain race-specific conditioning.
