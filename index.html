Key Stimulus

Compromised runs every 2–3 weeks: either your HYROX class or interval session becomes a circuit of run + stations (e.g. 3×500 m run + 20 wall balls + 40 m lunge). This is the core Block B adaptation.

HYROX Class by Phase
📗 Early weeks — Technique ▼
Show up and move well. Runs should feel easy (6:00–6:30/km). "Can I do 8 rounds and survive?"
📙 Mid weeks — Pace Awareness ▼
Push runs toward target HYROX pace. Stations should hurt, but runs should feel "controlled fast." "Can I hold this pace when legs are cooked?"
📕 Late weeks — Race Sim ▼
Target runs at exact race pace. If you deviate ±10 s/km, note it as data for race strategy. "This run IS the race."

Why This Works
🧠 Program Design Decisions ▼
Minimum viable dose: 2 runs + 1 squat + 1 metcon gives enough exposure to maintain running, strength, and conditioning without turning every week into a recovery problem.
Modular blocks: Rotating emphasis blocks lets you progress one quality without peaking everything at once. Don't run the same block for more than 10–12 weeks — the variation itself is part of the stimulus.
Classes as anchors: Classes are fixed stressors. Self-programming fills gaps, not competes. This prevents you from doubling up hard sessions on adjacent days.
Quality filter for HYROX class running: Class running counts toward volume, but only replaces an interval run when you intentionally target pace on every split.
Velocity squats in HYROX blocks: Submaximal load (55–60% 1RM) with fast bar speed preserves neuromuscular power with less residual fatigue than heavy slow reps — meaning you can run Runna's full plan without compromise.
Skill before strength: Bodyweight and gymnastics skill work belongs early in the session while CNS is fresh. After heavy squats or a long run, movement quality degrades and so does skill acquisition.
Oly block caution: Technical lifting has high neural demand. Running intensity and metcon volume need to drop so CNS resources go toward skill and power expression in the lifts.
Training Tips
⏱️ Recovery Separation ▼
24+ hours between hard endurance and heavy strength yields better strength adaptations. If you must stack on the same day, do strength before endurance and fuel aggressively in between.
Run club → next-day squats: Fine if squats stay lighter (RPE ≤7). Heavy build days are better 48 hrs after a hard interval session.
HYROX metcon → next-day squats: Light squats (3×3 RPE 7) are manageable. Avoid heavy builds — wait 48 hrs.
Limit to 2–3 genuinely hard sessions per week (e.g. interval run, HYROX class, and one heavy strength day). Everything else should be easy.
🍌 Fueling Around Concurrent Sessions ▼
Stacking endurance + strength same day: Fuel hard between sessions. Residual glycogen depletion from a run will tank your strength. Get carbs + protein in the gap.
Example: Interval run (6×600 m) → 200 kcal carb snack + electrolytes + 30 min rest → Thursday light squats is OK. Thursday heavy squats is risky without adequate refuelling.
Fatigue-first squat days (Z2 jog then squats): Eat light before the jog (e.g. banana), then fuel properly for the main squat lift. You want pre-fatigued legs, not a glycogen-depleted engine.
🏃 Race Pace Anchor ▼
Estimate your HYROX pace by Week 5–6. Run 3×1 km at a hard-but-controlled effort. Average the splits. Add 15–25 s/km to get your realistic HYROX race pace target.
Example: You average 4:58/km unsupported → target HYROX race pace ≈ 5:13–5:23/km → use 5:15/km as your interval anchor in Block B.
Your interval sessions should be ~10–15 s/km faster than race pace to build a speed reserve. In peak weeks, run repeats at or near race pace to lock it in.
Revisit every 3–4 weeks. If unsupported pace drops, your race pace anchor drops too.
🛑 Form Gates — When to Stop a Set
Squat: Stop if knees cave inward, butt tucks under, or bar drifts forward. Rest 2–3 min. Don't grind through degraded reps — they reinforce bad patterns.
Lunge / Step-up: Stop if front knee drifts well past toes, trailing knee bounces off the floor, or torso collapses forward.
Interval run: If stride shortens and cadence drops late in a rep, dial back pace by 10–15 s/km. Finish the session strong rather than accumulating garbage reps.
HYROX metcon: If wall balls lose target height, or farmer carry lists severely to one side, finish the round then reassess pacing for the next one.
For fatigue-resistance setups (Z2 → squats, HYROX → runs): keep load at 60–75% 1RM. Use volume, not weight, to create the stimulus.
📅 Taper Week (Race Week) ▼
Philosophy: Maintain quality of movement and pace. Slash volume by 40–50%. You should feel sharp, not stale.
Mon: 2–3×400 m at race pace. Long easy warm-up + cool-down. No metcon.
Tue: 3×3 squat at RPE 6 (light, move fast). Light accessories only — no heavy carries or step-ups. Optional 15 min easy Z2 bike if legs feel flat.
Wed: Easy jog 20–30 min. Station drills (SkiErg pacing, wall ball rhythm) at 60% effort.
Thu: Rest or 15 min easy jog + stretching.
Fri: 1–2×800 m at race pace (controlled, not all-out). Optional: 3 rounds of 500 m run at race pace + 1 station at 70% effort + easy transition walk.
Race day: Warm-up jog, station drills, go.
🔗 Multi-Race Strategy ▼
A-Race + B-Race: Use the B-Race as a fitness checkpoint, not all-out effort. Save peak pacing strategy for the A-Race.
Suggested structure: Weeks 1–6 Build → Week 7 Light taper for B-Race → Week 8 Post-B-Race deload (50% volume) → Weeks 9–11 Peak for A-Race → Week 12 Taper.
No race booked? A HYROX sim every 4–6 months in Block C or B is enough to test fitness and maintain race-specific conditioning.
